We have decided to use the online plotly tool for the purpose of result visualization. Plotly is an online analytics and data visualization tool that provides an integrated graphical user interface for importing and analysing input data.

Result Visualization

  • Result 1 :

    • Time comparison between MapReduce and Sequential Paradigm

    We have achieved is significant drop in running time by implementing the GA+FTS prediction model in distributed/parallel manner over normal sequential manner. MapReduce job usually splits the input data-set into independent chunks which are processed by the map tasks. The framework sorts the output of the mapper, which are then provided as an input to the reduce tasks. When the number of individuals are less, sequential manner takes less time as compared to MapReduce manner. As the number of individual increases, MapReduce paradigm has upper edge.

  • Result 2 :

    • MSE Vs. Individuals

    Initially when the number of generations are lower the mean square error is higher, however as the number of generations increases the value of MSE becomes lower until a point of convergence is reached. This means that with calculation of each iterative generation, the results obtained from our forecasting models become more accurate.

  • Result 3 :

    • Actual values vs. Predicted values for Rainfall

    The accuracy would be evident by using actual data set and comparing the same with our predicted values. As the number of generations is small, MSE value is high and hence the predicted values are not that accurate when compared to actual values. As the number of generations increase, the MSE values goes down (until it reaches convergence) there by increasing the accuracy of the predicted data.

  • Result 4 :

    • MSE Vs. Generations

    The convergence point is one of the most important evaluation parameters in our project. The set of chromosome that has the lowest MSE value will be the best solution that can be used to forecast the rainfall for that particular year. The two major trends observed are, when the number of intervals in the universe of discourse are increased the value of MSE tends to decrease. Secondly while generating fuzzy logical relationships, as the order increases MSE will decrease.
